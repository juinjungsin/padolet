쉬는시간 타이머 음악 자산

이 폴더에 다음 두 파일을 배치하세요. 파일이 없으면 타이머의 음악 토글이 자동으로 비활성화됩니다.

- `break-music.mp3` — 쉬는시간 동안 반복 재생할 배경 음악 (loop)
- `ding.mp3` — 타이머가 0에 도달할 때 한 번 재생할 알림음

권장 스펙
- 형식: MP3 (브라우저 호환성을 위해 권장)
- 길이: break-music은 1~3분 정도가 적당 (자동 loop)
- 볼륨: -16 LUFS 내외로 정규화
